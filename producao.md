## 🚀 Como Usar os Scripts
1️⃣ Verificar se está pronto:
```bash
python verificar_deploy.py
```

2️⃣ Criar pacote completo:
```bash
python criar_pacote_producao.py
```
3️⃣ Ou deploy rápido:
```bash
python deploy_rapido.py
```

## 📦 Exemplo de Uso Completo
```bash
python verificar_deploy.py
```
